# Collapse↑

*Every version of you exists right now.*  
*Calm. Brave. Clear. Present.*  
*All equally real. All waiting.*

-----

Collapse↑ is a contemplative web application that teaches quantum physics principles through direct experience rather than theory. You are not reading about the observer effect — you are the observer.

A particle field exists in superposition. Multiple versions of you drift, unresolved, in quantum blur. When you observe one — when you choose, breathe, and collapse — it becomes real. Not because it was always there. Because you measured it.

This is the same physics your nervous system runs on.

-----

## How it works

1. **Intro** — A particle field introduces superposition, observation, and collapse through experiential steps
1. **Choose** — Eight states of being exist in quantum superposition, each drifting, blurred, unmeasured
1. **Observe** — Your tap collapses the field. One version becomes real
1. **Breathe** — Three breath cycles encode the chosen state into the body. Each exhale sounds a crystallisation tone
1. **Repeat** — Return to the field. The particle remembers. Identity forms through accumulated attention

-----

## The science underneath

- **Quantum superposition** — systems exist in multiple states until observed
- **Wave function collapse** — observation selects one outcome from all possibilities
- **Neuroplasticity** — repeated observation patterns rebuild neural architecture
- **Identity formation** — what you consistently observe in yourself becomes what you are

## Technical

Built as a zero-dependency progressive web app. No frameworks. No build step. Pure browser.

**Stack**

- Vanilla JavaScript (ES6+)
- HTML5 Canvas for particle physics
- Web Audio API for drone, collapse, and crystallisation tones
- CSS animations for superposition motion
- Service Worker for offline support

**Deploy**

```
# Clone the repo
git clone https://github.com/coherence-nikolai/colapsar-hacia-arriba

# No build needed — serve the files directly
# GitHub Pages: push to main, enable Pages in Settings
```

**Files**

- `index.html` — structure and styles
- `app.js` — particle engine, breath sequencer, audio, state machine
- `data.js` — steps, states, imagination prompts (EN + ES bilingual)
- `service-worker.js` — offline caching
- `manifest.webmanifest` — PWA configuration

**Browser support**  
Optimised for iOS Safari and Chrome mobile. Web Audio API required for sound.
